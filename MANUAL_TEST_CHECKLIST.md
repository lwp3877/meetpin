## ✅ 1단계: 회원가입 페이지 테스트 (CRITICAL)

### 로컬 테스트: http://localhost:3001/auth/signup

**시각적 검증**:
- [ ] 페이지 로딩이 완전히 완료됨 (스피너가 사라짐)
- [ ] 베타 배너가 상단에 표시됨
- [ ] 모든 입력 필드가 보임 (8개 항목):
  - [ ] 이메일 입력 필드
  - [ ] 비밀번호 입력 필드
  - [ ] 비밀번호 확인 필드
  - [ ] 닉네임 입력 필드
  - [ ] 연령대 선택 드롭다운
  - [ ] 체크박스 4개 (이용약관, 개인정보처리방침, 베타 테스트 동의, 마케팅 수신 동의)
  - [ ] 회원가입 버튼
  - [ ] 소셜 로그인 버튼 3개 (카카오, 구글, 네이버)

**기능 검증**:
1. **이메일 입력 테스트**:
   - [ ] 유효하지 않은 이메일 입력 → 에러 메시지 표시됨
   - [ ] 유효한 이메일 입력 → 에러 없음

2. **비밀번호 검증**:
   - [ ] 6자 미만 입력 → "최소 6자 이상" 에러
   - [ ] 비밀번호 확인 불일치 → "비밀번호가 일치하지 않습니다" 에러
   - [ ] 일치하는 비밀번호 → 에러 없음
   - [ ] 비밀번호 강도 인디케이터 작동

3. **닉네임 검증**:
   - [ ] 1자 입력 → "최소 2자 이상" 에러
   - [ ] 21자 입력 → "최대 20자" 에러
   - [ ] 2-20자 입력 → 에러 없음

4. **체크박스 검증**:
   - [ ] 필수 체크박스 미체크 → 가입 불가
   - [ ] 필수 체크박스 체크 → 가입 가능
   - [ ] 마케팅 수신 동의는 선택사항

5. **소셜 로그인 버튼**:
   - [ ] 카카오 버튼 클릭 → OAuth 페이지로 리다이렉트 (또는 "설정되지 않음" 메시지)
   - [ ] 구글 버튼 클릭 → OAuth 페이지로 리다이렉트 (또는 "설정되지 않음" 메시지)
   - [ ] 네이버 버튼 → 비활성화 상태 확인

6. **실제 회원가입 테스트**:
   - [ ] 모든 필드 입력
   - [ ] 회원가입 버튼 클릭
   - [ ] 로딩 상태 표시됨
   - [ ] 성공 시 /map으로 리다이렉트
   - [ ] 실패 시 명확한 에러 메시지

**브라우저 콘솔 확인**:
- [ ] F12 개발자 도구 열기
- [ ] Console 탭에서 에러 없음 확인
- [ ] Network 탭에서 API 호출 성공 확인

## ✅ 3단계: 지도 페이지 테스트 (CRITICAL)

### 로컬 테스트: http://localhost:3001/map

**시각적 검증**:
- [ ] 카카오 지도가 완전히 로드됨 (서울 지역)
- [ ] 지도 컨트롤이 작동함 (확대/축소/이동)
- [ ] 상단 필터 버튼 3개 (전체, 술🍻, 운동💪, 기타✨)
- [ ] 하단 "방 만들기" 버튼
- [ ] 네비게이션 바 (지도, 채팅, 프로필)

**지도 기능 검증**:
1. **지도 인터랙션**:
   - [ ] 마우스 드래그 → 지도 이동됨
   - [ ] 확대/축소 버튼 → 줌 레벨 변경됨
   - [ ] 내 위치 버튼 → 위치 권한 요청 (또는 현재 위치로 이동)

2. **필터 기능**:
   - [ ] "전체" 클릭 → 모든 방 표시
   - [ ] "술🍻" 클릭 → 술 카테고리만 표시
   - [ ] "운동💪" 클릭 → 운동 카테고리만 표시
   - [ ] "기타✨" 클릭 → 기타 카테고리만 표시

3. **마커 표시**:
   - [ ] 지도에 마커가 표시됨 (있는 경우)
   - [ ] 마커 클릭 → 방 상세 정보 표시
   - [ ] 부스트된 방 → 특별한 스타일 표시

**데이터 확인 (개발자 도구)**:
```javascript
// Console에서 실행:
fetch('/api/rooms?bbox=37.4,126.8,37.7,127.2')
  .then(r => r.json())
  .then(d => console.log('방 개수:', d.data.rooms.length))
```
- [ ] 방 개수가 0개인지 확인
- [ ] 0개인 경우 → **CRITICAL ISSUE**: 샘플 데이터 추가 필요
