## 1. 프로그래밍이 뭔가요?

### 🤔 간단히 말하면...
- **프로그래밍**: 컴퓨터에게 "이런 일을 해줘"라고 명령을 내리는 것
- **코드**: 컴퓨터가 이해할 수 있는 언어로 작성한 명령서
- **웹사이트**: 인터넷을 통해 사람들이 볼 수 있는 페이지

### 🏠 집 짓기에 비유하면...
- **HTML**: 집의 뼈대 (기둥, 벽, 문, 창문)
- **CSS**: 집의 꾸미기 (페인트, 인테리어, 가구)
- **JavaScript**: 집의 기능 (전기, 수도, 엘리베이터)

## 2. 밋핀은 어떤 프로젝트인가요?

### 🎯 밋핀의 목적
**지도에서 사람들이 만날 수 있는 플랫폼**을 만드는 프로젝트입니다.

### 🔍 주요 기능들
1. **지도 표시**: 카카오맵을 사용해서 지도를 보여줍니다
2. **모임 만들기**: 사용자가 원하는 장소에 모임을 만들 수 있습니다  
3. **채팅 기능**: 모임에 참여한 사람들끼리 채팅할 수 있습니다
4. **결제 시스템**: 모임을 더 잘 보이게 하는 "부스트" 기능을 살 수 있습니다

### 🛠️ 사용된 기술들 (쉽게 설명)
- **Next.js**: 웹사이트를 만드는 도구 (마치 집을 짓는 기계처럼)
- **TypeScript**: JavaScript의 업그레이드 버전 (실수를 줄여주는 도구)
- **Tailwind CSS**: 디자인을 쉽게 만드는 도구 (미리 만들어진 스타일)
- **Supabase**: 데이터를 저장하고 관리하는 서비스 (클라우드 창고)

## 3. 필요한 프로그램 설치하기

### 📥 설치 순서

#### 3.1 Git 설치 (코드 다운로드 도구)
1. [https://git-scm.com](https://git-scm.com) 방문
2. "Download" 버튼 클릭
3. 다운로드된 파일 실행하여 설치
4. 설치 중 모든 옵션은 기본값으로 두기

#### 3.2 Node.js 설치 (JavaScript 실행 환경)
1. [https://nodejs.org](https://nodejs.org) 방문
2. "LTS" 버전 다운로드 (안정적인 버전)
3. 다운로드된 파일 실행하여 설치
4. 설치 중 모든 옵션은 기본값으로 두기

#### 3.3 pnpm 설치 (패키지 관리 도구)
1. **Windows**: 명령 프롬프트(cmd) 또는 PowerShell 열기
2. **Mac**: 터미널 열기
3. 아래 명령어 입력:
```bash
npm install -g pnpm
```

#### 3.4 VS Code 설치 (코드 편집기)
1. [https://code.visualstudio.com](https://code.visualstudio.com) 방문
2. "Download" 버튼 클릭
3. 다운로드된 파일 실행하여 설치

### ✅ 설치 확인하기
명령 프롬프트/터미널에서 다음 명령어들을 실행해보세요:

```bash
git --version    # Git 버전이 나오면 성공
node --version   # Node.js 버전이 나오면 성공  
pnpm --version   # pnpm 버전이 나오면 성공
```

## 4. 프로젝트 실행해보기

### 📂 프로젝트 다운로드
1. 원하는 폴더에서 명령 프롬프트/터미널 열기
2. 아래 명령어 입력:
```bash
git clone https://github.com/your-username/meetpin.git
cd meetpin
```

### 📦 필요한 패키지 설치
```bash
pnpm install
pnpm approve-builds
```

### ⚙️ 환경 설정 (중요!)
1. 프로젝트 폴더에 `.env.local` 파일 생성
2. 아래 내용 복사해서 붙여넣기:

```env
# 개발 모드 설정 (처음에는 이것만 있으면 됩니다!)
NEXT_PUBLIC_FORCE_MOCK=true

# 사이트 주소 설정
SITE_URL=http://localhost:3000
```

### 🚀 서버 실행하기
```bash
pnpm dev
```

성공하면 이런 메시지가 나옵니다:
```
✓ Ready in 3s
- Local: http://localhost:3000
```

### 🎉 결과 확인하기
1. 웹브라우저 열기
2. 주소창에 `http://localhost:3000` 입력
3. 밋핀 웹사이트가 보이면 성공!

## 6. 간단한 수정 방법

### 🎨 색깔 바꾸기
1. `src/app/globals.css` 파일 열기
2. CSS 색상 코드 찾아서 변경하기

```css
/* 예시: 주색상을 파란색으로 바꾸기 */
.primary-color {
  background-color: #3b82f6; /* 파란색 */
}
```

### ✏️ 텍스트 바꾸기
1. `src/app/page.tsx` 파일 열기
2. 원하는 텍스트 찾아서 수정하기

```typescript
// 예시: 제목 바꾸기
<h1>밋핀에 오신 것을 환영합니다!</h1>
// ↓ 이렇게 바꾸기
<h1>새로운 친구를 만나보세요!</h1>
```

### 🖼️ 이미지 바꾸기
1. 새 이미지를 `public/` 폴더에 저장
2. 코드에서 이미지 경로 수정

```typescript
// 예시: 로고 이미지 바꾸기
<img src="/logo.png" alt="로고" />
// ↓ 이렇게 바꾸기  
<img src="/new-logo.png" alt="새 로고" />
```

## 7. 문제 해결 방법

### 🚨 자주 발생하는 문제들

#### 문제 1: "포트가 이미 사용 중입니다"
```bash
# 해결방법: 포트 강제 종료
npx kill-port 3000
pnpm dev
```

#### 문제 2: "패키지를 찾을 수 없습니다"
```bash
# 해결방법: 패키지 재설치
pnpm store prune
pnpm install
```

#### 문제 3: "빌드 에러가 발생했습니다"
```bash
# 해결방법: 종합 검사 실행
pnpm repo:doctor
```

#### 문제 4: "웹페이지가 하얗게 나옵니다"
1. 브라우저 콘솔 확인 (F12 키 누르기)
2. 빨간색 에러 메시지 확인
3. `.env.local` 파일 설정 재확인
